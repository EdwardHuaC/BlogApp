# DearDiary
[DearDiary](https://cryptic-lowlands-34818.herokuapp.com/) is full-stack website application.     
It has been deployed on Heroku: https://cryptic-lowlands-34818.herokuapp.com/

### Technologies
- JavaScript ES6
- Bootstrap 4
- Node.js
- Express.js
- MongoDB and Mongoose
- Deployment with Heroku and MongoDB Atlas
